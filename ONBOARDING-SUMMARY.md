## Important Project Constraints

### Brand Compliance ⚠️
- **NO clip art** — Never hand-draw SVGs of real objects (containers, trucks)
- **AI from real photos** — Use Gemini/Claude image gen from client's actual photos
- **Pre-built assets** — Always check `/assets/` first before creating new graphics

### Design Standards
- **Source of truth:** GitHub repository (repo is authoritative)
- **Stitch + Claude Design:** One-way GitHub re-sync (repo is source)
- **Landing pages:** Must pass 16-element conversion checklist
- **Brand colors:** Forest Green (#2D5016) primary, Louisiana State Green accent

### Workflow Rules
- Branch naming: `feature/name` or `content/article-name`
- All PRs require review before merge
- Commit messages must be descriptive and reference related issues
- Test locally before marking ready
