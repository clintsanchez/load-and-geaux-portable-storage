## System Details

### Color Palette (Aligned Across Both Systems)
| Use | Hex | RGB | Accessible |
|-----|-----|-----|-----------|
| **Primary** | #2D5016 | 45, 80, 22 | ✅ 8.2:1 contrast |
| **Accent** | #004D40 | 0, 77, 64 | ✅ Hover state |
| **Background** | #FAF9F7 | 250, 249, 247 | ✅ Warm, not sterile |
| **Text** | #1F2937 | 31, 41, 55 | ✅ 13.8:1 contrast |
| **Body** | #6B7280 | 107, 114, 128 | ✅ 4.8:1 contrast |

**Dark mode:** Full palette defined with proper contrast ratios.

### Typography System
**Display:** Poppins Bold/ExtraBold (700/800) — headlines, big numbers  
**Body:** Inter Regular/SemiBold (400/600) — paragraphs, captions  
**Utility:** Monospace — pricing, specifications, data

**Scale:** H1 (40px) → H2 (32px) → H3 (24px) → Body (16px) → Small (14px)

### Accessibility

✅ WCAG AA Compliant
- 4.5:1 contrast ratio for all text
- Visible focus states on interactive elements

- Semantic HTML structure

✅ Fully Implemented
- CSS custom properties for theme switching
- Both `prefers-color-scheme` media query and manual toggle support
- All components tested in light and dark
- Proper contrast maintained in both modes
