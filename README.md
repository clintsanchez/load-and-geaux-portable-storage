# Load and Geaux Portable Storage

Professional website, content, and marketing systems for Load and Geaux Portable Storage — portable storage and shipping container rentals in Baton Rouge, Louisiana.

## Quick Start

### Project Directories
```
📁 brand/              — Brand guidelines, logos, visual identity
📁 docs/               — Documentation, guides, project context
📁 blog/               — Blog posts and content clusters
📁 website/            — Website code and structure
📁 marketing/          — Marketing materials, templates, campaigns
📁 seo/                — SEO strategy, keyword research, optimization
📁 social/             — Social media content and calendars
📁 assets/             — Photography, infographics, graphics
📁 memory/             — Project memory and context tracking
```

### Key Files
- **CLAUDE.md** — Project instructions and context for Claude Code
- **DESIGN.md** — Complete design system (colors, typography, components, dark mode)
- **DESIGN-QUICK-REFERENCE.md** — One-page designer/developer cheat sheet
- **brand-style-guide.md** — Brand voice, tone, and usage guidelines
- **PHASE-2-ACTION-ITEMS.md** — Website audit findings and prioritized tasks
- **docs/AUDIT-PHASE-2-REPORT.md** — Comprehensive website audit with stack detection
- **docs/DESIGN-SYSTEM-OVERVIEW.md** — High-level design system philosophy and approach
